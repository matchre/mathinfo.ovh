On utilise la méthode .index() appliquée à une chaîne de caractères qui renvoie l'indice du premier caractère de la chaîne identique à celui donné en argument de la méthode. 


##----- Définition des Fonctions -----##
def majuscules(texte):
    """ Entrées : texte est une chaîne de caractères
        Sorties : renvoie texte dans lequel le premier caractère est remplacé par le caractère
            correspondant."""
    minus="abcdefghijklmnopqrstuvwxyz"
    majus="ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    lettre = texte[0]
    indice = minus.index(lettre)
    return majus[indice] + texte[1:]

Dans le codage ASCII décimale, la lettre majuscule 'A' est représentée par la valeur 65 et la lettre majuscule 'a' est représentée par la valeur 97. Il suffit donc de soustraire 32 à la représentation décimale d'une lettre minuscule pour obtenir la majuscule correspondante.
On rappelle que les fonctions Python qui permettent cette conversion sont :

  • ord() du caractère vers l'ASCII décimal ;
  • chr() de l'ASCII décimal vers le caractère.

##----- Définition des Fonctions -----##
def majuscules(texte):
    """ Entrées : texte est une chaîne de caractères
        Sorties : renvoie texte dans lequel le premier caractère est remplacé par le caractère
            correspondant."""
    valeur = ord(texte[0])
    return chr(valeur-32) + texte[1:]


##----- Programme principal -----##
f = open('dico.txt', 'r')
g = open('dico3.txt', 'w')

for mot in f:
    g.write(majuscules(mot))


##----- Fermeture des fichiers -----##
f.close()
g.close()

Avoir un deuxième liste de stockage du nombre de chaque caractère semble indispensable. Pour cela, l'instruction [0]*26 est utile... 

##----- Définition des Fonctions -----##
def compte_lettres(nom_fichier):
    """ Entrées : nom_fichier est une chaîne de caractères correspondant à un nom de fichier
        Sorties : renvoie la liste des occurences de chaque lettre (minuscule) du fichier."""
    liste = list('abcdefghijklmnopqrstuvwxyz')
    fsource = open(nom_fichier, 'r')
    occurences = [0]*26
    
    for ligne in fsource:
        for carac in ligne:
            if carac in liste:
                indice = liste.index(carac)
                occurences[indice] += 1
    fsource.close()
    return occurences
