Compléter la définition de la fonction compter() qui prend en paramètres une chaîne de caractères et un caractère. Cette fonction doit compter le nombre d'occurences (le nombre de fois où apparaît) du caractère dans la chaîne.

Voici un exemple d'affichage où le programme demande à l'utilisateur d'entrer une phrase (ci-dessous 'Ceci est un exemple.') et un caractère (ci-dessous 'e') puis il retourne le nombre d'occurences du caractère dans la phrase : 

Entrez une phrase : Ceci est un exemple.
Entrez un caractère : e
Il y a 5 "e" dans la phrase.

Ce programme doit évidemment faire appel à la fonction compter(). Il est par contre inutile de vérifier les exceptions.

						La fonction compter() prend les entrées de
						l'utilisateur comme paramètres et retourne un nombre qui sera affiché.
					

					  
					

						

							La fonction compter() se
							conçoit en trois temps :
						

						
    
							
  • On parcourt chaque caractère de la chaîne ;
    • 
							
  • Lorsque le caractère est carac, on compte 1
								de plus ;
    • 
							
  • A la fin du parcours, on renvoie la valeur obtenue.
    • 
						
	
					

					  
					

						

##----- Définition des Fonctions ----- ##
def compter(chaine, carac):
    """ Entrees : chaine est une chaine de caracteres et carac est un caractere
        Sorties : le nombre d'occurences de carac dans chaine."""
    nombre = 0                  # Variable locale à la fonction qui stocke le nombre d'occurences
    for lettre in chaine:       # On parcourt les lettres du texte
        if lettre == carac:
            nombre += 1
    return nombre               # La fonction retourne le nombre d'occurences


##----- Programme principal ----- ##
texte = input('Entrez une phrase : ')
lettre = input('Entrez un caractere : ')
n = compter(texte, lettre)
print('Il y a {} "{}" dans la phrase.'.format(n, lettre))

##----- Définition des Fonctions ----- ##
def remplacer(texte, carac1, carac2):
    """ Entrees : texte est une chaine de caracteres, carac1 et carac2 sont des un caracteres
        Sorties : texte dans lequel chaque caractère carac1 est remplacé par carac2."""
    chaine = ''                 # La chaîne qui sera retournée
    for lettre in texte:       	# On parcourt les lettres du texte
        if lettre == carac1:
            chaine += carac2
        else:
            chaine += lettre
    return chaine               # La fonction retourne le texte modifié


##----- Programme principal ----- ##
texte = input('Entrez une phrase : ')
c1 = input('Entrez un caractère : ')
c2 = input('Entrez un autre caractère : ')
print('La phrase modifiée est :', remplacer(texte, c1, c2))

##----- Définition des Fonctions ----- ##
def espacer(texte):
    """ Entrees : texte est une chaine de caracteres
        Sorties : texte dans lequel un espace a été inséré entre chaque caractère."""
    chaine = ''                 # La chaîne qui sera retournée
    for lettre in texte:
        chaine += lettre        # On ajoute chaque caractère de texte
        if lettre != ' ':
            chaine += ' '       # On ajoute un espace si le caractère n'en était pas un
    return chaine               # La fonction retourne le texte modifié


##----- Programme principal ----- ##
texte = input('Entrez une phrase : ')
print('La phrase modifiée est :', espacer(texte))

def inverser(texte):
    """ Entrees : texte est une chaine de caracteres
        Sorties : écriture "inversée" de texte."""
    chaine = ''                 	# La chaîne qui sera retournée
    for lettre in texte:
        chaine = lettre + chaine	# La concaténation 'Rage'+'Cou' donne 'RageCou'
    return chaine


##----- Programme principal ----- ##

    print('Cette phrase n\'est pas un palindrome')

##----- Définition des Fonctions ----- ##
def inverser(texte):
    """ Entrees : texte est une chaine de caracteres
        Sorties : écriture "inversée" de texte."""
    return texte[::-1]               # Parcours à l'envers de la chaîne de caractères


##----- Programme principal ----- ##
texte = input('Entrez une phrase : ')
reponse = inverser(texte)
print('La phrase modifiée est :', reponse)

def narcisse(n):
    """ Entrees : n est un entier naturel
        Sorties : renvoie True si n est narcissique, c'est-à-dire un entier égal à la somme
            de ses p chiffres chacun élevé à la puissance p. Renvoie False sinon."""
    chaine = str(n)
    p = len(chaine)
    total = 0					# Somme des p (chiffres puissance p)
    for chiffre in chaine:
        total += int(chiffre)**p
	
    if total == n:
        return True
    else:
        return False
