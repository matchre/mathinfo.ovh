Dans cet exercice, on peut utiliser le paramètre sep de la fonction print().
Attention au nombre d'espaces devant la 1ère étoile à afficher. Heureusement, un espace est un caractère comme un autre... 

##----- Définition des Fonctions ----- ##
def etoile3(n):
    """ Entrees : n est un entier strictement positif
        Sorties : affichage d'un triangle constitué de n à 1 étoiles, séparées par des espaces."""
    motif ="* "
    for i in range(n):
        print(" "*i, end="")        # D'abord 0, puis 1, puis 2, puis... espaces
        print(motif*(n-i))          # On imprime le motif n fois puis (n-1) puis (n-2) puis ...


##----- Programme principal ----- ##
n = int(input('Entrez un entier strictement positif : '))
etoile3(n)

C'est toujours le même principe. Attention aux espaces devant les étoiles.
Devant la 1ère étoile, il y a 3 espaces. Or l'utilisateur a entré le nombre 4. On peut intuitivement en déduire que si l'utilisateur entre le nombre n, il y aura (n-1) espaces devant la 1ère étoile. 

##----- Définition des Fonctions ----- ##
def etoile4(n):
    """ Entrees : n est un entier strictement positif
        Sorties : affichage d'un triangle constitué de 1 à n étoiles, séparées par des espaces."""
    motif ="* "
    for i in range(1, n+1):
        print(" "*(n-i), end="")    # D'abord (n-1), puis (n-2), puis (n-3), puis... espaces
        print(motif*i)              # On imprime le motif 1 fois puis 2 puis 3 puis ...


##----- Programme principal ----- ##
n = int(input('Entrez un entier strictement positif : '))
etoile4(n)

Dans la fonction login(chaine1, chaine2),

  • on extrait le 1er caractère de ch1 avec ch1[0] ;
  • puis on renvoie une concaténation des caractères grâce à l'opérateur «+».
C'est la même idée qui permet de définir la fonction masquer(). 

##----- Définition des Fonctions ----- ##
def login(chaine1, chaine2):
    """ Entrees : deux chaînes de caractères
        Sorties : renvoie la 1ère lettre de chaine1 et chaine2 complet, séparés par un point."""
    return chaine1[0] + '.' + chaine2


def masquer(chaine):
    """ Entrees : une chaîne de caractères
        Sorties : renvoie la 1ère lettre et la dernière lettre de chaine, séparées par trois étoiles '***'."""
    return chaine[0] + '***' + chaine[len(chaine)-1]

nom = input('Quel est votre nom ? ')
prenom = input('Quel est votre prénom ? ')

print('Votre nom masqué est : {}'.format(masquer(nom)))
